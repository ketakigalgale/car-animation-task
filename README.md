# 🚗 Scroll-Driven Hero Section Animation

A high-performance, scroll-synced hero animation built using **Next.js, TypeScript, and GSAP**, focused on delivering a smooth, premium user experience similar to modern product landing pages.

## 🎯 Objective

To recreate a **scroll-based interactive hero section** with smooth animations, staggered text effects, and scroll-linked motion while maintaining high performance and clean code structure.

---

## ✨ Features

* 🔤 **Letter-by-letter headline animation (staggered reveal)**
* 🚗 **Scroll-synced car movement using GSAP ScrollTrigger**
* ✨ **Smooth interpolation for natural motion**
* 📊 **Animated statistics with staggered entry**
* 🎯 **Scroll-driven interaction (not time-based autoplay)**
* ⚡ **Optimized performance using transform-based animations**
* 💎 **Premium, smooth UI experience**

---

## 🛠️ Tech Stack

* **Framework:** Next.js (App Router)
* **Language:** TypeScript
* **Animation Library:** GSAP (ScrollTrigger)
* **Styling:** Tailwind CSS / CSS

---

## ⚙️ How It Works

### 🔹 Initial Load Animation

* Headline text appears using staggered animation
* Statistics fade in one by one with delay

### 🔹 Scroll-Based Animation

* Animation is linked to scroll position using GSAP ScrollTrigger
* Car movement is controlled by scroll progress (not time)
* Smooth easing ensures natural motion

### 🔹 Performance Optimization

* Used `transform` properties (translate, scale)
* Avoided layout reflows for smooth rendering
* Efficient animation handling with GSAP

## 🚀 Setup & Installation

Clone the repository:

```bash
git clone https://github.com/ketakigalgale/car-animation-task.git
cd car-animation-task
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```
